Introduction

This project was developed for NewEra, a national newspaper in Namibia, as part of my work at Green Enterprise Solutions. It involved building both a dashboard for managing subscriptions and an e-paper system, which digitized the newspaper experience for users. The project was developed using .NET6, jQuery, and Bootstrap, delivering a modern and interactive experience.

Project Background

NewEra's e-paper solution needed to replicate the physical newspaper experience in a digital format, including a flipbook interface that allows users to browse the newspaper page by page. My role in the project focused on implementing the e-paper functionality and backend integrations to fetch and display the digital papers for users.

Key Features

  • Flipbook interface for the e-paper, providing a real-world reading experience.
  • User profile management, allowing users to update their personal information and view subscription details.
  • Automated email reminders for subscription renewals.
  • Backend integration for displaying e-papers fetched from the server.
  • Analytics functionality within the dashboard to track views, category-specific performance, and subscription revenue per month.

Implementation

As part of this project, I focused primarily on the e-paper side, pulling the newspapers from the backend and displaying them to users in a flipbook format. I also developed the profile functionality, which enables users to manage their account details and view subscription information. Additionally, I handled backend integrations to send email notifications reminding users to renew their subscriptions before they expired.

Although I did not work directly on payment integration, I contributed to the dashboard by adding analytics features to track total views, views per category, and monthly revenue from subscriptions.

Future Enhancements

  • Enhanced analytics capabilities for more detailed reporting on user engagement.
  • Integration of new subscription models and payment gateways.
  • Improved user interface for profile management and e-paper browsing.
  • Mobile app integration to bring the e-paper experience to a broader audience.

Conclusion

This project played a crucial role in modernizing how NewEra engages with its readers, offering a seamless digital reading experience. The combination of a user-friendly e-paper system and robust subscription management tools has transformed how users access and interact with content. My involvement in this project was a great learning experience in developing scalable and interactive web solutions.
